Options

Options helps to customize the plugin and create different types of portfolios according to your specific needs.

1. columns

Number of columns in a row 

default: 4
options: integer

2. mode

Type of portfolio 

default: "image"
options: "image", "video"

3. margin

space between each box 

default: 2
options: integer

4. caption

If true, allow caption on each portfolio 

default: true
options: boolean (true / false)

5. captionMode

Type of caption. Captions are derived from the image's/video's data attributes. Note: when it comes to small devices it may not display the text 

default: "icon"
options: "title", "text", "icon", "title|text"

6. background

Background color of caption 

default: "rgba(255,0,0,0.7)"
options: string

7. textColor

Font color of caption title and text 

default: "#fff"
options: string

8. captionFit

If true, allow caption to fit to the box size 

default: false
options: boolean (true / false)

9. animateCaption

If true, allow animation on caption title, text or icon (It will slide in from the top) 

default: false
options: boolean (true / false)

10. animation

Type of animation on the portfolio 

default: "pf-fade"
options: "pf-zoomin", "pf-zoomout", "pf-flip-left", "pf-flip-right", "pf-flip-up", "pf-flip-down", "pf-flip-rotate", "pf-fade", 
         "pf-popup", "pf-slide-up", "pf-slide-down", "pf-slide-left", "pf-slide-right", "pf-move-left", "pf-move-right", "pf-hide"

11. clickEventOn

Element that click event is added on 

default: "box"
options: "box", "caption", "icon"

12. onclickAction

Action which is triggered after the click 

default: "popup"
options: "popup", "url", "none"

13. linkTarget

Defines the way of opening an URL. Note: only used if onclickAction: "url" 

default: "new"
options: "same", "new"

14. backdropColor

Background color of popup backdrop 

default: "rgba(0,0,0,0.7)"
options: string

15. popupSize

Size of the popup modal. Note: only used if onclickAction: "popup" 

default: "medium"
options: "small", "medium", "large"

16. controls

If true, allow all controls on popup window 

default: true
options: boolean (true / false)

17. closeController

If true, allow close button on popup window 

default: true
options: boolean (true / false)

18. navController

If true, allow navigation buttons on popup window 

default: true
options: boolean (true / false)

19. fixedNavigation

If true, allow navigation buttons to stay fixed on popup window 

default: true
options: boolean (true / false)

20. backdropClose

If true, allow to close the popup window by click on backdrop 

default: true
options: boolean (true / false)

21. closeOnEsc

If true, allow to close the popup window by press on Escape button 

default: true
options: boolean (true / false)

22. responsive

If true, allow to resize the portfolio grid auto. Note: Useful if you need to use fixed number of columns 

default: true
options: boolean (true / false)
