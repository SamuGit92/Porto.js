Introduction

Porto.js is a great jQuery plugin for adding customized image or video portfolios to your website. It only supports for the sites that build up with the bootstrap framework. It helps you to display your projects, client logos, photography, or any other pictures or videos of your choice in a much more productive way.

  • Fully responsive - will adapt to any device

  • Portfolios can contain images or videos

  • Captions can be added on to portfolio

  • Uses CSS transitions for image and video animation

  • Images and Videos can be viewed in a popup window

  • Small file size, simple to implement

  • Browser support: Firefox, Chrome, Opera

  • Number of customized options

If you read our options page we have provided instructions in step by step to use the various features and functionality of this plugin.

How to install

1. Including Files

First and most important thing is, that including jQuery library and bootstrap files. Then link the Porto.js CSS and Javascript file. 

<!-- jQuery library -->
<script src="js/jquery-1.11.2.js"></script>

<!-- bootstrap Javascript file -->
<script src="js/bootstrap.min.js"></script>

<!-- porto Javascript file -->
<script src="js/jquery.porto.js"></script>

<!-- bootstrap CSS file -->
<link  href="css/bootstrap.min.css" rel="stylesheet" />

<!-- porto CSS file -->
<link  href="css/jquery.porto.css" rel="stylesheet" />

2. HTML Structure

For each portfolio need to create one <div class="porto"> element with <div class="pf"> tags inside it as much as you need to display boxes. Inside each of these divs should contain data attributes to hold the information about portfolio as follows. Note that only "data-pf-image" is necessary and others can be added as your choice. Check the examples for more information about HTML structure.

<div class="porto">
  <div class="pf" data-pf-image="images/1.jpg" data-pf-title="Title" data-pf-caption="Caption" data-pf-onclick="URL"></div>
  <div class="pf" data-pf-image="images/2.jpg" data-pf-title="Title" data-pf-caption="Caption" data-pf-onclick="URL"></div>
  <div class="pf" data-pf-image="images/3.jpg" data-pf-title="Title" data-pf-caption="Caption" data-pf-onclick="URL"></div>
  <div class="pf" data-pf-image="images/4.jpg" data-pf-title="Title" data-pf-caption="Caption" data-pf-onclick="URL"></div>
</div>

3. Initialization

Call .porto() on <div class="porto">. If you have different class name make sure to call the function with that class. 

$(document).ready(function () {
  $('.porto').porto();
});

Check the examples for more information on customization.
